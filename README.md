Self-use color system and utils.

# Installation

How to install

```bash
# with npm
npm install @canisminor/colors --save-dev

# with yarn
yarn add @canisminor/colors --dev
```

## Development

Install dependencies

```bash
$ yarn
```

Start the dev server

```bash
$ yarn start
```

Build documentation

```bash
$ yarn docs:build
```

Run test

```bash
$ yarn test
```

Build library via `father-build`

```bash
$ yarn build
```
